# Skills from A Smart Bear

Claude Code skills that implement frameworks and algorithms from the
[A Smart Bear](https://longform.asmartbear.com) blog and the book
[Hidden Multipliers](https://hiddenmultipliers.com), curated by Jason Cohen.

This archive is a snapshot of every skill published at the site below. Each
skill is a single, self-contained `SKILL.md` file in its own folder.

## Where this came from

- **Website & documentation:** https://skills.asmartbear.com/
- **How to install:** https://skills.asmartbear.com/install/
- **Source on GitHub:** https://github.com/asmartbear/asb-skills

## Installing

Unzip this archive into your Claude Code skills folder, then restart Claude
Code (or run `/context`) to pick them up:

```bash
unzip asb-skills.zip -d ~/.claude/skills/
```

Each skill lands in its own `~/.claude/skills/<name>/SKILL.md`. Delete any you
don't want — every skill stands on its own.
